# Delta: Approval Requests

## ADDED Requirements

### Requirement: Ошибка отзыва после начала согласования

Система SHALL возвращать отдельную причину отказа, если пользователь пытается отозвать заявку после начала согласования.

#### Scenario: API возвращает отдельный error code

- GIVEN заявка находится в процессе согласования
- AND текущий пользователь является автором заявки
- WHEN пользователь пытается отозвать заявку
- THEN система отклоняет операцию с кодом `409`
- AND возвращает error code `APPROVAL_ALREADY_STARTED`
- AND статус и данные заявки не изменяются

#### Scenario: UI показывает понятное сообщение

- GIVEN API вернул error code `APPROVAL_ALREADY_STARTED`
- WHEN UI обрабатывает ответ
- THEN UI показывает сообщение «Заявку нельзя отозвать: согласование уже началось»
- AND UI обновляет текущие данные заявки
